## node.Js : var, let, const 차이
1. var
 - 변수가 내용이 변경된 상태로 2번 이상 선언이 되어도 문제없이 실행된다. (마지막으로 선언된 변수로 저장)
 - 코드량이 많아질 경우 변수명을 중복으로 사용하지 않는다는 보장이 없어 크나큰 문제를 일으킬 수 있다.
 - clean code를 위해 사용하지 않는다.

2. let
 - 변수 선언을 오로지 1개만 가능하다
 - let로 선언한 변수는 재할당이 가능하다.
 - 변수 선언 시 사용하기 좋다

3. const
 - 변수 선언을 오로지 1개만 가능하다
 - let로 선언한 변수는 재할당이 불가능하다 (java static같은 존재?)
 - 상수 선언 시 사용하기 좋다
